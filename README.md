# Snake

Einfaches Snake Game.

## Funktionen
- Hauptmenü
- Highscore anzeigen


## Installation
1. Klone dieses Repository:

    git clone https://github.com/RinChise/gamesnake.git


2. Installiere die Abhängigkeiten:
    
    pip install -r requirements.txt


3. Starte die Anwendung:

    python3 main.py

## Voraussetzungen
- Python 3.12
